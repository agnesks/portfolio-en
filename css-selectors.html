Project Overview

This microlearning experience is based on a blended learning project I developed in 2020 at ELTE University. The project was an HTML & CSS basics module within a digital literacy course for humanities students. The course combined in-person introductory lessons with self-paced video tutorials, code-alongs and practical exercises which led to a final website project.

Course outcomes showed that learners struggled to apply CSS selectors in practice, especially during early project work. This often led to confusion, frequent errors, and frustration when their code didn’t behave as expected. This microlearning module focuses on addressing these recurring issues by targeting common beginner mistakes and supporting learners in developing a more practical understanding of selectors through interaction. The module also introduces debugging early, helping learners diagnose why their code doesn’t work.

The module focuses on foundational selectors (element, class, id) to establish a clear mental model of targeting. More complex patterns, such as descendant selectors, are introduced later to avoid cognitive overload. The module follows foundational HTML and CSS modules (HTML Basics, Tables, Forms, What is CSS?) and prepares learners for advanced CSS topics such as complex selectors, layout, and more complex debugging.

The module is developed in Articulate Storyline to support interactive, problem-based learning where learners actively identify and fix mistakes.

⚠️ Problems:

  • Beginners often confuse CSS selectors, especially class and id
  • They struggle to apply selectors in real scenarios
  • They lack effective debugging strategies

🎯 Goal:

  • Reduce early frustration in coding tasks by addressing common beginner mistakes
  • Develop basic debugging and problem solving skills

🧠 Learning Objectives:

  • Learners will be able to differentiate between element, class and id selectors and apply them correctly
  • Learners will be able to identify and fix simple selector-related issues

📈 Learning Outcomes:

  • Reduced confusion when working with selectors
  • Improved ability to fix simple CSS issues independently

👥 Target Audience:

  • Beginners who already have a basic understanding of HTML and how CSS works but are fairly new to the concept of CSS selectors

🛠️ Tools & Approach:

  • Articulate Storyline
  • Scenario-based interactions: quizzes, interactive coding and debugging exercises

About the original course

The original blended learning course was developed and delivered in 2020 at Eötvös Loránd University in Budapest, Hungary. It was an HTML & CSS basics module within a digital literacy course for humanities students. The course combined in-person introductory lessons with self-paced video tutorials, code-alongs and practical exercises which led to a final website project. The e-learning materials were hosted in Canvas LMS, which was also used to manage assignment submissions and grading.

Outcomes:

All students successfully created their own website and completed the course. However, several insights emerged. Beginners often struggled to grasp core CSS concepts, which created early friction and reduced motivation for some students. In addition, many lacked debugging and problem-solving skills, which caused them to get stuck during exercises. The e-learning module described above was designed to address these challenges.
