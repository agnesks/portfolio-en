CSS Selectors: Target the Right Element

Microlearning Module
Canvas
CodePen
Quizzes

Articulate Storyline
Context:

This microlearning module is based on a previous HTML+CSS blended learning course. Course outcomes showed that learners struggled to apply CSS selectors in practice. This often led to confusion and frustration when their code didn’t behave as expected. This module addresses these issues by targeting common beginner mistakes and helping learners develop a more practical understanding of selectors. It also introduces debugging early, enabling learners to diagnose why their code doesn't work.

What I did:

Designed an interaction-focused e-learning module.

Outcomes:

By the end of the module learners will be able to apply selectors correctly and fix simple selector-related issues.
